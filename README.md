# 项目说明
- 扑克概率计算 
- 工作室网站 [www.poker-x-studio.com](www.poker-x-studio.com)

## 一 功能说明:
  总体是扑克类游戏概率的统计计算,分别为

    - 德州扑克-统计公共牌部分牌型出现概率
    - 无限德州扑克-2张手牌和5张公共牌之间,出现一对的概率统计
    - Blackjack胜率统计,玩法策略统计,下注策略统计
    - Baccarat胜率统计
    - Baccarat押注策略[待完成]
