# 项目说明
- 扑克概率计算 
- 工作室网站 [www.poker-x-studio.com](www.poker-x-studio.com)

## 一 功能说明:
  总体是扑克类游戏概率的统计计算,分别为

    - 德州扑克-统计公共牌部分牌型出现概率
    - 无限德州扑克-2张手牌和5张公共牌之间,出现一对的概率统计
    - Blackjack胜率统计,玩法策略统计,下注策略统计
    - Blackjack庄家爆牌率
    - Baccarat胜率统计
    - Baccarat押注策略[待完成]
    - Baccarat闲家/庄家点数分布概率
    - Plo4四张奥马哈 胜率计算
    - Plo5五张奥马哈 胜率计算

## 二 技术栈
    - Go语言开发
    - svgo包生成svg图
    - svg转jpeg图
